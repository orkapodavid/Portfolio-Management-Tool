# Run Dev Server

// turbo-all

1. Sync dependencies:
```bash
cd d:\GitHub\Portfolio-Management-Tool\reflex-ui-starter
uv sync
```

2. Initialize Reflex (first time only):
```bash
cd d:\GitHub\Portfolio-Management-Tool\reflex-ui-starter
uv run reflex init
```

3. Start the dev server:
```bash
cd d:\GitHub\Portfolio-Management-Tool\reflex-ui-starter
uv run reflex run
```

4. Open browser at http://localhost:3000
