## Running the Demo

```bash
cd reflex_ag_grid/examples/demo_app
uv run reflex run
# Open http://localhost:3000
```

## Document Structure

Each documentation file includes:
- **Requirement Description**: What feature this page demonstrates
- **AG Grid Enterprise Feature**: The AG Grid APIs and props used
- **Live Demo Link**: Link to the demo page route
- **Code Example**: Key code snippets from the implementation
- **How to Implement**: Step-by-step guide for your own app
